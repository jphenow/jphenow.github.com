Backtick Code Blocks

Jul 26th, 2011

With the backtick_codeblock filter you can use Github’s lovely back tick syntax highlighting blocks. Simply start a line with three back ticks followed by a space and the language you’re using.

Syntax

``` [language] [title] [url] [link text]
code snippet
```

Example (plain)

```
$ sudo make me a sandwich
```

Example With Syntax Highlighting a Caption and Link

``` ruby Discover if a number is prime http://www.noulakaz.net/weblog/2007/03/18/a-regular-expression-to-check-for-prime-numbers/ Source Article
class Fixnum
  def prime?
    ('1' * self) !~ /^1?$|^(11+?)\1+$/
  end
end
```

This is a nice, lightweight way to add a highlighted code snippet. For features like titles and links you’ll want to look at the codeblock or include_code liquid tags.
